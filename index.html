경영진 요약

무인 키오스크가 가져올
운영 방식의 전환

무인 키 보관함 키오스크는 단순 자동 장치가 아니라, 차량 서비스 운영 구조를 바꾸는 인프라입니다. 고객 편의, 직원 업무 효율, 관리자 원격 제어를 동시에 실현합니다.

고객 만족도 향상

24시간 키 보관·출고가 가능해지고, 차량번호·PIN 입력만으로 대기 시간이 줄어듭니다.

운영 효율성 증대

키 관리 업무가 줄어 직원은 정비·상담에 집중하고, 관리자는 원격 모니터링으로 장애에 대응합니다.

브랜드 신뢰성 강화

전자락·디지털 로그로 분실·오류를 최소화하고, TLS·이중 인증 보안 설계로 신뢰를 제공합니다.

확장 가능한 구조

키오스크는 HTTPS 아웃바운드 연결로만 동작해 현장 네트워크 설정이 간단하고, 다지점 확장이 쉽습니다.

문제 정의

키 관리의 병목, 왜 자동화가 필요한가?

차량 대여·정비에서는 키 인수인계가 늘 병목입니다. 무인 시간에는 키를 맡길 방법이 없고, 바쁜 시간에는 열쇠 찾는 대기가 길어집니다. 수동 관리는 분실 위험과 관리 공수를 동시에 키웁니다.

인수인계 지연

직원·고객 동선이 겹치며 평균 대기 시간이 길어집니다. 대기 스트레스는 서비스 만족도를 떨어뜨립니다.

분실·오류 위험

종이 기록과 수동 보관은 실수와 분실을 항상 동반합니다. 누가 언제 키를 가져갔는지 추적이 어렵습니다.

무인 시간 서비스 공백

관리자가 상주하지 않는 시간에는 키를 맡기거나 찾을 방법이 없어 서비스가 멈춥니다.

관리자 상주 부담

보관함을 직접 열어 키를 꺼내고 반납받는 방식은 관리 공수를 키우고 확장을 가로막습니다.

제안 구조

보관함 제어와 클라우드 서버 연동 구조

키오스크는 차량번호·비밀번호 입력 후 RS485/USB IO보드로 20개 보관함을 개별 제어합니다. 센서 데이터와 이용 이력은 HTTPS REST API / WebSocket으로 클라우드에 전송되며, 관리자 웹 대시보드에서 실시간 모니터링합니다.

UI/UX 시나리오

차량키 맡기기·찾기 시나리오

고객은 차량번호만으로 빈 보관함에 키를 맡기고, 정비 완료 후 문자로 받은 PIN으로 키를 찾습니다. 정비사는 차량번호 검색으로 보관함을 열어 작업 후 다시 보관할 수 있습니다.

차량키 맡기기

  1. 차량번호 입력
  2. 빈 보관함 자동 배정
  3. 보관함 문 열림
  4. 키 투입 · 문 닫힘
  5. 접수 완료 — 서버에 이력 저장

차량키 찾기

  1. 차량번호 입력
  2. 문자로 받은 PIN 입력
  3. 본인 확인
  4. 보관함 개방
  5. 키 수령 완료

정비사 메뉴

차량번호 검색으로 해당 보관함을 열어 키를 가져가고, 작업 완료 후 다시 보관합니다. 고객용 PIN은 서버에서 발급됩니다.

상단 진행 표시와 큰 버튼으로 초보자도 쉽게 이용합니다. 키오스크와 대시보드는 동일 색상 규칙(초록: 대여 가능 · 주황: 대여 중 · 빨강: 열림)으로 상태를 표시합니다.

FAQ

검토 단계에서 가장 자주 나오는 질문

무인 시간에도 키를 맡기고 찾을 수 있나요?

네. 관리자 상주 없이 24시간 무인 운영됩니다. 고객은 차량번호로 키를 맡기고, 정비 완료 후 문자로 받은 PIN을 입력해 키를 찾습니다. 모든 인수인계 이력은 서버에 저장됩니다.

보관함 제어와 이상 감지는 어떻게 하나요?

RS485 기반 IO보드 또는 USB 릴레이보드로 20구 전자락을 개별 제어합니다. 각 도어의 센서가 열림/닫힘과 미닫힘 오류를 감지해 상태를 갱신하며, 이상 상태는 관리자 웹 대시보드에 실시간 표시되고 원격으로 도어 테스트가 가능합니다.

네트워크·보안 구성은 어떻게 되나요?

키오스크는 HTTPS REST API / WebSocket으로 클라우드 서버와 통신합니다. 아웃바운드 연결로만 동작해 NAT 뒤에서도 현장 네트워크 설정이 간단합니다. 모든 API 호출은 TLS 암호화와 장치 고유 토큰으로 보호되며, 관리자 강제 개방 시 이중 인증을 거칩니다.

도입 비용과 일정은 어떻게 되나요?

1단계는 1대 시제품 제작과 현장 PoC로 시작합니다. UI/UX 프로토타입과 시나리오 시뮬레이션 검증은 완료된 상태이며, 세부 견적과 일정은 미팅에서 설치 환경·수량 기준으로 산정해 드립니다. 양산 규모는 PoC 결과를 근거로 협의합니다.
